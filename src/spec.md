# Specification

## Summary
**Goal:** Build a responsive One Dot Digital marketing website (Home, About, Services, Testimonials, Contact) using the provided logo, with cohesive branding and tasteful scroll-based animations.

**Planned changes:**
- Create a multi-section page (or equivalent navigation experience) with sections: Home, About Us, Services, Testimonials, Contact Us.
- Add a top navigation that scrolls/jumps to each section, includes a clear primary CTA (“Get in touch”/“Contact”), and a footer that repeats the same navigation links.
- Apply a consistent visual theme (colors/typography/spacing/components) aligned with the uploaded One Dot Digital brand image; avoid a generic blue+purple gradient look unless it matches the logo styling.
- Use the uploaded image “Add a heading.jpeg” as the logo in the header (linked to top) and as a prominent hero visual, rendering crisply across devices.
- Implement scroll-based motion (mixed option): section reveal animations for Home/About, at least one subtle parallax-like effect, and Services card hover/focus interactions plus scroll-triggered animation; respect reduced-motion preferences.
- Add the About Us content with the exact “Who Are We?” heading and provided paragraph copy in a polished layout.
- Implement Services as a responsive grid of 7 cards with the provided descriptions and “Includes” sublists.
- Implement Testimonials with all provided entries (name + business/descriptor + text) in a responsive grid or carousel with consistent formatting.
- Implement Contact Us showing the exact email, phone number, and Instagram URL; repeat these in the footer with clickable mailto/tel and an external link opening in a new tab.
- Add additional locally stored, marketing-themed imagery (beyond the logo) to support the hero/section transitions.

**User-visible outcome:** Visitors can smoothly navigate a polished, animated One Dot Digital website, read about the agency, browse all services and testimonials, and contact the agency via clickable email/phone/Instagram links.
